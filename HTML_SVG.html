Scalable Vector Graphics (SVG)

  1. SVG is used to define graphics for the Web
  2. The HTML < svg> element is a container for SVG graphics.
  3. SVG has several methods for drawing paths, boxes, circles, text, and graphic images.
  4. SVG is a language for describing 2D graphics in XML.
  5. SVG is XML based, which means that every element is available within the SVG DOM. You can attach JavaScript event handlers for an element.
