HTML "table" tag

  1. An HTML table is defined with the < table> tag.
  2. Each table row is defined with the < tr> tag. A table header is defined with the < th> tag. By default, table headings are bold and centered. A table data/cell is defined with the < td> tag.

  3. Basic HTML Table

    Firstname Lastname Age
    Jill Smith 50
    Eve Jackson 94
    John Doe 80
    Note: press "ctrl+u" to see the code
  4. The < td> elements are the data containers of the table. They can contain all sorts of HTML elements; text, images, lists, other tables, etc.
  5. If you do not specify a border for the table, it will be displayed without borders. A border is set using the CSS border property.
    table, th, td { border: 1px solid black; }

    Remember to define borders for both the table and the table cells.

  6. If you want the borders to collapse into one border, add the CSS border-collapse property;
    table, th, td { border: 1px solid black; border-collapse: collapse; }
  7. Cell padding specifies the space between the cell content and its borders. If you do not specify a padding, the table cells will be displayed without padding. To set the padding, use the CSS padding property
    th, td { padding: 15px; }
  8. By default, table headings are bold and centered. To left-align the table headings, use the CSS text-align property:
    th { text-align: left; }
  9. To make a cell span more than one column, use the colspan attribute:
    < th colspan="2">Telephone< /th>
    make a cell span more than one row, use the rowspan attribute: < th rowspan="2">Telephone:< /th>
  10. To add a caption to a table, use the < caption> tag: 
            < table style="width:100%">
        < caption>Monthly savings< /caption>
        < tr> 
          < th>Month< /th>
          < th>Savings< /th>
        < /tr>

    Note: The < caption> tag must be inserted immediately after the < table> tag.
  11. To define a special style for a special table, add an id attribute to the table
