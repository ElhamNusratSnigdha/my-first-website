Break Line tag/"br" & Preformatted tag/"pre"

  1. The HTML < br> element defines a line break or a new line without starting a paragraph.
  2. This is an empty tag, which means that it has no end tag.
  3. < p>This is< br>a paragraph< br>with line breaks.< /p>
  1. The HTML < pre> element defines preformatted text
  2. The pre tag preserves both spaces and line breaks
  3. < pre>
    My Bonnie lies over the ocean.
    My Bonnie lies over the sea.
    My Bonnie lies over the ocean.
    Oh, bring back my Bonnie to me.
    < /pre>
