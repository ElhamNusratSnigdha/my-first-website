HTML Video & Audio

  1. The HTML5 < video> element specifies a standard way to embed a video in a web page.
  2. The HTML5 < audio> element specifies a standard way to embed audio in a web page.
  3. Before HTML5, a video & audio could only be played in a browser with a plug-in (like flash).
  4.           <--Internal Video-->
            < video src="/img/video.mp4" width="300" controls>
            Your browser does not support the video tag.
            < /video>

          <--External Video-->
            < iframe src="https://www.youtube.com/embed/wg-WvrSTf7k" frameborder="0">
            < /iframe>

  7. How it Works
    • The controls attribute adds video controls, like play, pause, and volume.
    • It is a good idea to always include width and height attributes. If height and width are not set, the page might flicker while the video loads.
    • The element allows you to specify alternative video files which the browser may choose from. The browser will use the first recognized format.
    • The text between the < video> and < /video> tags will only be displayed in browsers that do not support the < video> element.
    • To start a video automatically use the "autoplay" attribute.
      NOTE: The autoplay attribute does not work in mobile devices like iPad and iPhone.
    • HTML5 defines DOM methods, properties, and events for the < video> element. This allows you to load, play, and pause videos, as well as setting duration and volume. There are also DOM events that can notify you when a video begins to play, is paused, etc.

            var myVideo = document.getElementById("video1"); 
            function playPause() { 
              if (myVideo.paused) 
                myVideo.play(); 
              else 
                myVideo.pause(); 
            } 
  
            function makeBig() { 
                myVideo.width = 560; 
            } 
            
            function makeSmall() { 
                myVideo.width = 320;
